# Live Editor Examples (with on-premise server)

## prepare

```bash
yarn
```

## simple demo:

```bash
yarn simple
```

Open http://localhost:9000 in browser.

## toolbar and custom blocks, boxes and custom text style, etc...

```bash
yarn custom
```

## custom mention

```bash
yarn mention
```

## custom date box

```bash
yarn date
```

input `dd` then insert a date box.


## custom label box

```bash
yarn label
```

input `ll` then insert a label box.
